# TwitchStream

This project was created using the Simplr Create command

```bash
npm init @simplr-wc
```

## Installation

```bash
npm install twitch-stream
```

```html
<script src="./src/twitch-stream.js" type="module"></script>

<twitch-stream></twitch-stream>
```

```bash
npm run start
```
